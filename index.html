Minha jornada

Sou um profissional de TI com 2 anos de experiência, especializado em desenvolvimento e análise de sistemas. Minha formação inclui linguagens de programação orientada a objetos (POO) como Java e PHP, combinadas com JavaScript e práticas de Clean Code para criação de designs responsivos usando HTML, CSS e Frameworks como Bootstrap. Meu foco é entregar soluções eficientes, organizadas e inovadoras, alinhadas às necessidades do negórcio e às expectativas dos usuários.

Ao longo da minha trajetória, trabalhei no diagnóstico e solução de problemas técnicos em sistemas como Linux e Windows, manuntenção de hardware e software e otimização de processos para garantir contínuo e com a aplicação da melhores práticas em desenvolvimento, desde os princíos de Clean Code até o trabalho colaborativo em equipe. Meu objetivo é contribuir para projetos tecnológicos impactantes que gerem valor real para organizações e usuários.
